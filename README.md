    利用历史数据构建指数预测模型及短线选股模型，早盘自动进行计算，对今日指数涨跌方向进行预测，以及相关挑选相关短线股。构建的数据包括A股市场历史数据,美股市场历史数据，相关期货数据，以及其他文本类消息等。
每个交易日造成9点25后利用当天的交易数据以及历史数据进行指数预测和计算，大概9点27左右给出当天的指数预测结果及股票池。如下：
